# Turtle Race – Python Projekt

Tento projekt je jednoduchá grafická hra vytvořená pomocí knihovny **turtle** v Pythonu.
Cílem projektu bylo procvičit práci s více objekty, náhodností, smyčkami a základní herní logikou.

Projekt vychází z výukového kurzu a slouží čistě jako **trénink programování**.

---

## Jak hra funguje

Uživatel si na začátku zvolí barvu želvy, na kterou si vsadí.
Následně se spustí závod, ve kterém se želvy pohybují **náhodnou rychlostí** směrem k cíli.

Program:

1. Vytvoří několik želv různých barev
2. Umístí je na startovní pozici
3. Zeptá se uživatele na tip vítěze
4. Spustí závod pomocí `while` smyčky
5. Vyhodnotí vítěze a vypíše výsledek
