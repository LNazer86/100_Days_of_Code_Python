# Mile to Kilometer Converter (Tkinter)

Jednoduchá desktopová aplikace v Pythonu, která převádí míle na kilometry pomocí grafického rozhraní v knihovně **Tkinter**.

## Jak aplikace funguje
- Uživatel zadá hodnotu v mílích do vstupního pole.
- Po stisknutí tlačítka **Calculate** se hodnota přepočítá na kilometry.
- Výsledek se okamžitě zobrazí v aplikaci.

## Použité technologie
- Python
- tkinter

## Spuštění
1. Ujisti se, že máš nainstalovaný Python.
2. Spusť soubor:
```
python main.py
```
