## Turtle Crossing Game (Python & Turtle)

Jednoduchá arkádová hra ve stylu Frogger vytvořená v Pythonu pomocí knihovny turtle.

## Funkce
- Ovládání želvy klávesami nahoru / dolů
- Náhodně generovaná auta
- Postupné zrychlování hry po každém levelu
- Počítání aktuálního levelu
- Game Over při kolizi s autem

## Ovládání
- Pohyb nahoru: šipka ↑  
- Pohyb dolů: šipka ↓  

## Spuštění
python main.py

## O projektu
Projekt byl vytvořen jako součást kurzu  
**100 Days of Code: The Complete Python Pro Bootcamp by Dr. Angela Yu**.  
Cílem bylo procvičit práci s třídami, dědičností, detekcí kolizí, pohybem objektů a herní smyčkou.
