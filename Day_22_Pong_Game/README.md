# Pong Game (Python & Turtle)

Jednoduchá klasická hra Pong vytvořená v Pythonu pomocí knihovny turtle.

## Funkce
- Ovládání dvou pádel (W/S a šipky nahoru/dolů)
- Odrazy míčku od stěn a pádel
- Postupné zrychlování míčku
- Počítání skóre pro oba hráče
- Reset míčku po gólu

## Ovládání
- Levý hráč: W / S  
- Pravý hráč: šipka nahoru / dolů  

## Spuštění
python main.py

## O projektu
Projekt byl vytvořen jako součást kurzu **100 Days of Code: The Complete Python Pro Bootcamp by Dr. Angela Yu**.  
Cílem bylo procvičit práci s třídami, dědičností, herní smyčkou, detekcí kolizí a logikou hry pomocí knihovny turtle.
