# Snake Game – Python OOP Projekt

Tento projekt je klasická hra **Snake (Had)** vytvořená v Pythonu pomocí knihovny **turtle** a objektově orientovaného programování.

Projekt vznikl v rámci výukového kurzu a slouží jako **praktický trénink OOP, herní logiky a práce s více soubory**.

---

## Jak hra funguje

Hráč ovládá hada pomocí šipek na klávesnici.  
Cílem je sbírat potravu, prodlužovat hada a vyhnout se:

- nárazu do stěny
- nárazu do vlastního těla

Za každé sebrané jídlo se zvyšuje skóre.

---

## Funkce hry

- pohyb hada pomocí kláves
- náhodně generovaná potrava
- růst hada po snědení jídla
- počítání skóre
- detekce kolizí se stěnou
- detekce kolizí s vlastním tělem
- zobrazení „Game Over“

---

## Jak spustit hru

1. Stáhněte nebo naklonujte repozitář  
2. Otevřete projekt v IDE (PyCharm, VS Code, …)  
3. Spusťte soubor `main.py`  
4. Ovládejte hada pomocí šipek  
5. Snažte se dosáhnout co nejvyššího skóre
