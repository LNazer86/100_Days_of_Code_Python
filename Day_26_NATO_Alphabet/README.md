# NATO Phonetic Alphabet Converter

Jednoduchý Python program, který převede zadané slovo na seznam fonetických kódových slov podle NATO abecedy.

## Jak program funguje
- Načte data z CSV souboru pomocí knihovny **pandas**.
- Vytvoří slovník ve tvaru `{písmeno: kódové_slovo}`.
- Uživatel zadá slovo.
- Program vypíše seznam odpovídajících NATO kódů pro každé písmeno.

## Použité technologie
- Python
- pandas

## Spuštění
1. Nainstaluj knihovnu pandas:
```
pip install pandas
```
2. Spusť soubor:
```
python main.py
```
