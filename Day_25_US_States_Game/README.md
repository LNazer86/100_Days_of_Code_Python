# U.S. States Game

Jednoduchá geografická hra vytvořená pomocí knihovny **Turtle** a **Pandas**. Cílem je postupně uhádnout všech 50 států USA a zobrazit je na mapě.

## Jak hra funguje
- Zadáváš názvy států do vstupního okna.
- Pokud je odpověď správná, název se zobrazí na správném místě na mapě.
- Hra sleduje počet správných odpovědí.
- Příkaz **Exit** ukončí hru a uloží neuhádnuté státy do souboru `missed_states.csv`.

## Použité technologie
- Python  
- turtle  
- pandas  

## Spuštění
1. Ujisti se, že máš nainstalovaný Python.
2. Nainstaluj knihovnu pandas:
   ```
   pip install pandas
   ```
3. Spusť soubor:
   ```
   python main.py
   ```
